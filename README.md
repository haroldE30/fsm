# fsm
Prerequisites:
1. Java 1.6+
2. Maven 3.3+

To run:
1. Open terminal
2. Clone repository
3. Change directory to fsm/
4. type: mvn install
   (this will download all jar files needed for the application and will run the test cases included)
5. log will display on the console or you can open the file log/FSMApp.log
